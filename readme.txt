MADO:A DoucWorks plugin
本プラグインは、富士ゼロックス株式会社 「Docuworks」がインストールされた環境のみで動作します。

1. 概要
  DocuWorks文書の1ページ目の一部をワンクリックで拡大して表示するためのツールです。表示箇所は、MADO.Hにハードコーディングしています。

2. ライセンス条項
　このソフトウェアは、富士ゼロックス株式会社の「DocuWorks(TM) Development Tool Kit 9.10」を用いて作成されています。

　このソフトウェアは、MITライセンスの元に提供されます。

3. インストール
　Release/MADO.dllをDocuworksのpluginsフォルダにコピーしてください。

4,使用方法
　Docuworks Desk上でファイルを選択した状態で、本プラグインのアイコンをクリックしてください。

5,コンパイル方法
「追加のインクルードディレクトリ」にPLUGINSPI,XDWAPIのインクルードディレクトリを追加してください。
「追加の依存ファイル」にXDWAPI.libを追加してください。

6,動作環境
 Windows10
 Docuworks 9以上がインストールされていること。

7,開発環境
 Visual Studio 2019 Community
